01 Jan 2014

One of my university assignments required creating and rendering a animation in POVray. POVray is raytracing software where you describe a scene using text and it renders it for you. If you describe success scenes (parameterised on time) then you can render all the frame and join them into an animation.

I chose to animate a Newton’s Cradle, but there was extreme scope creep. I modelled what one might consider to be Newton’s writing desk. Each object took quite a lot of time to model individually. For example I’m quite sure that the tree was fractally generated.

The animation was well received but I don’t think it was the most visually impressive. It was a good lesson to work smarter and not harder, but I still thoroughly enjoyed the time that I spent on this.
